## Installation

[Use npm](https://docs.npmjs.com/cli/install).

```sh
npm install gulp-tasks --save-dev
```

[Use yarn](https://yarnpkg.com/en/docs/usage). ([Installation](https://yarnpkg.com/lang/en/docs/install/#mac-tab))

```sh
yarn add gulp-tasks --dev
```
